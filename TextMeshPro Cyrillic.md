To generate font asset with cyrillic:
1. Character Set - Unicode Range(Hex)
2. Character Sequence (Hex) - 0-7f,400-4ff

0-7f - latin
400-4ff - cyrillic

All unicode charts: http://www.unicode.org/charts/
